# How to Set Up BMAD

## English

### What is BMAD?

BMAD (Breakthrough Method for Agile AI-Driven Development) is a framework that optimizes AI-assisted development with minimal human intervention. The goal: intent in, code changes out, with as few back-and-forth questions as possible.

Documentation: https://docs.bmad-method.org/explanation/quick-dev/

### Setup

**1. Create a branch:**
```bash
git checkout -b bmad
```

**2. Install BMAD (bleeding edge) in your project directory:**
```bash
npx github:bmad-code-org/BMAD-METHOD install
```

Follow the interactive setup dialog and choose the options that fit your project.

> Note: This version pulls directly from the `main` branch and may be unstable.
